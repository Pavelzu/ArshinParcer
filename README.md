# ArshinParcer
Проект предназначен для работы с порталом fgis.gost.ru . Последний не позволяет фильтровать СИ (средство измерения) по периодичности поверки. Единственный выход - перебирать карточки СИ и считывать оттуда информацию о поверке. Для автоматизации этой работы создан этот программный продукт.
